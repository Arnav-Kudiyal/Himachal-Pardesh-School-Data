# Himachal-Pardesh-School-Data
In this repository, I have used the affiliation Data of all private schools of Himachal which are affiliated to the Himachal Pardesh Board of School Education.

I have used Pandas in this project and performed all the steps of Data Wrangling to make my data Clean. The steps that I have done on this data are-
#### Collection of Data - Data is taken from the official site of the HPBOSE .(https://hpbose.org)
#### Accessing of Data - Data is carefully accesed to perform further steps.
#### Cleaning of Data - Data is cleaned by various Pandas Functions.
#### Structuring of Data - Data is structured according to the Project need.
#### Enriching of Data - New Columns have been added to enhance the data quality.
#### Stroing of Data - Data is stroed into various .csv files according to the need. In this project files are divided according to the district of HP.

## Details -
In this project a data file of all the schools of the Himachal(Private) is given. The main objective is to filter out the schools which are senior sercondary and also have stream science in them for further processes and filter out schools according to the districts and save in different files.
